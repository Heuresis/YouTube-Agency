# Interview Prompt — Research Brief

## Cluster 1 — Topic + scope
1. Specific topic of this video?
2. One-sentence angle?
3. What awareness level?
4. Which belief from the 8-belief path does it install?

## Cluster 2 — VOC pull on topic
5. Pull 10+ verbatim audience phrases specifically about this topic.
6. Source each.

## Cluster 3 — Competitive scan
7. Name 3-5 competitor archetypes (anonymized) who cover this topic.
8. What angle does each take?
9. What's working / missing in each?
10. Where does this channel differentiate?

## Cluster 4 — Fact-check
11. List every numeric / specific claim the script will make.
12. Source each.
13. Flag any unverifiable claims as REMOVE or REFRAME.

## Cluster 5 — Beats
14. Hook hypothesis (3+ mechanisms in first 30s)?
15. BUT-THEREFORE arc?
16. 5-9 sectional beats?
17. Loop architecture (N loops, types, placements)?
18. Close + CTA?

## Synthesis
Produce the brief.
