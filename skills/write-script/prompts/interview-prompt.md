# write-script — interview mode (gather before you write)

> Fires when the gate fails (identity < 70 / audience < 70 / hook_retention_os < 60 / content_engine < 50) **or** when a script request is missing essentials. Ask only what's missing — never guess the things that decide positioning. Phrase every question so the creator can answer it in one message or voice memo, not in workspace jargon.

## Essentials (must have before writing)
1. **Working title / topic** — what's the one-line of this video?
2. **The exact viewer** — not a demographic. *Whose* situation/struggle/goal? (e.g., "agency owner at $2M tired of doing ops by hand").
3. **Desired action** — subscribe · watch-next · download the lead magnet · book a call · apply?
4. **The proof** — which real number(s)/result(s)/artifact(s) anchor it? (Trace each to a `workspace/PROOF-BANK.md` row; anything unverified becomes a `[CREATOR: confirm …]` slot — never invent one.)
5. **Length** — default 8–16 min long-form.

## Strategic (sharpens the positioning — the 80%)
6. **Awareness guess** — does the target viewer know they have the problem? know solutions exist? know *this creator's* solution? (Pulls from `workspace/STRATEGY.md` defaults if unanswered.)
7. **Sophistication** — how crowded is this exact topic on YouTube, and what is everyone already saying? (So the script picks how to be *different/better* — its ≥ 3 of the six differentiation paths.)
8. **The belief target** — what's the one belief this video installs, or the one thing everyone in the niche gets wrong?
9. **Differentiation angles** — any insight, story, or positioning the creator owns that competitors can't copy? (A counted dataset, a contrarian take, an operator-of-record vantage.)
10. **Video type** — case study · mechanism walkthrough · story/origin · teardown/live-diagnosis · listicle.
11. **CTA destination + the next video** (for the binge loop).

## If the creator can't answer 6–9
Make the reasonable assumption from the topic + `workspace/STRATEGY.md` / `workspace/PROOF-BANK.md` / `workspace/CALIBRATION.md`, **state the assumption in the output**, and proceed. Don't block the script on a sophistication questionnaire the creator wouldn't think in.

## Hand back
Once gathered, run `SKILL.md` Steps 1 → 5. Return the script + the reconciliation log + VOICE-CHECK markers + OPEN SLOTS.
