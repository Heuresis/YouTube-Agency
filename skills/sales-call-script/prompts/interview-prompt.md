# Interview Prompt — Sales Call Script

1. Offer + price + structure?
2. Audience dominant limiting belief + transformation type?
3. Top 8-10 objections from compartment 2?
4. Discovery framework preference (8-stage / FST / SPIN)?
5. Closing methodology preference?
6. Existing post-call cadence?

## Synthesis
Write 25-30 min script across opening / discovery / pitch / objection / close / post-call branches.
