# Templates

> Fill-in scaffolds. Every template is creator-agnostic; operator-specific variables are filled at execution time from `company.yaml` and the creator's compartments. Each file carries its own fill-in frontmatter block, structure spec, and verification gate.

## Scripts

| File | What it scaffolds | When to reach for it |
|---|---|---|
| `script-authority.md` | Long-form authority script — belief installation toward the back-end offer, 8–25 min | `/write-authority-script` |
| `script-reach.md` | Long-form reach script — broad-appeal contrast / contrarian structures, 4–10 min | `/write-reach-script` |
| `script-shorts.md` | 30–60 second Shorts script — compressed hook / body / CTA | `/write-shorts-script` |
