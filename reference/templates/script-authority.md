# Authority Script Template

> Long-form authority script template. Used for authority-mode videos that target the channel's ICP and convert via belief installation toward the back-end offer. Mode: authority. Target length: 8–25 minutes (1500–3000 words). Awareness target: solution-aware → most-aware (calibrated per video).

## Frontmatter

```yaml
title: <working title>
mode: authority
awareness_target: <solution-aware | product-aware | most-aware>
length_target_min: <8-25>
word_count_target: <1500-3000>
content_pillar: <pillar from compartment 5>
target_avd_pct: <typically 50-55>
hook_formula: <PAST | AID | Contrast | Statistic | Curiosity-Gap | Bound-Promise | Contrarian-Frame | Social-Proof-Stack>
beliefs_to_install: [<list of beliefs from the 8 standard or niche-calibrated stack>]
end_screen_target: <next video URL or playlist URL>
cta: <CTA description: lead-magnet / pinned-VSL / application / soft-subscribe>
```

## Section 0 — Cold open (0:00–0:08)

The Reptilian-brain decision happens here. Strong visual + audible hook within 0–8 seconds.

```
[VISUAL: <strong opening visual — close-up, on-screen text, surprising frame>]
[AUDIO: <energy-forward delivery>]

<HOOK SENTENCE — 1–2 lines that immediately deliver on the title-thumbnail-promise>
```

## Section 1 — Hook stack (0:08–0:30)

Layer 3+ retention mechanisms (INV-11). Choose from:

- Curiosity-gap (open-loop)
- Bound-promise (specific deliverable + time)
- Contrarian frame (niche-canonical belief flipped)
- Statistic shock (specific number that activates Limbic)
- Social-proof stack (3+ outcomes named)
- Pattern-interrupt (cut, music shift, on-screen super)

```
<MECHANISM 1: hook formula>
<MECHANISM 2: bound-promise OR contrarian frame>
<MECHANISM 3: open-loop set up>

[PATTERN INTERRUPT: cut / b-roll / super at 0:30 boundary]
```

## Section 2 — Re-hook + frame (0:30–1:30)

Re-state the video's promise. Set the frame. Establish credibility briefly. Loop is opened by 0:08; this section sustains it.

```
"Here's exactly what we're going to walk through, and why it matters."

<FRAME — 3–5 sentences that contextualize the topic for the awareness target>
<CREDIBILITY ANCHOR — 1–2 sentences of relevant operator standing>
<RE-PROMISE — what the viewer will leave with by the end>
```

## Section 3 — Section 1 (1:30–4:00)

First major teaching block. Use re-hook → story → lesson per-section pattern.

```
[RE-HOOK — pose a question that the section will answer]

<STORY OR EXAMPLE — 60–120 seconds; voice-of-viewer language; specific>

<LESSON — the structural insight extracted from the story>

[CALLBACK — close one curiosity-gap from earlier; open another]

[PATTERN INTERRUPT: cut at section transition]
```

## Section 4 — Section 2 (4:00–7:00)

Second major teaching block. Same per-section pattern. Pattern-interrupt cadence ≥ every 60s.

```
[RE-HOOK]

<TEACHING — mechanism / framework / case explanation>

<APPLICATION — how the viewer applies the teaching>

[CALLBACK + new loop opening]
```

## Section 5 — Section 3 (7:00–10:00)

Third major teaching block. Apply for: deeper mechanism, second-order implications, or counter-argument handling.

```
[RE-HOOK]

<DEEPER TEACHING or OBJECTION HANDLING>

<EVIDENCE or CASE-STUDY-EMBED>

[CALLBACK]
```

## Section 6 — Bridge to CTA (10:00–11:30)

Synthesis. Pull the teaching together. Set up the CTA without immediately pitching.

```
<SYNTHESIS — what the viewer now knows; identity-aspirational framing>

<TRANSITION — from teaching to next step>

"If you want to go deeper on this, here's what's next."
```

## Section 7 — CTA (11:30–end)

The CTA is calibrated to the awareness target:

- **Solution-aware target**: lead-magnet bridge or foundational-playlist routing
- **Product-aware target**: pinned-VSL reference + soft offer mention
- **Most-aware target**: application CTA + pinned-VSL link prominent

```
<CTA — 30–60 seconds>

[ON-SCREEN: link / next-video card / subscribe prompt]
[END-SCREEN: next video + playlist or pinned VSL]
```

## Production cues throughout

- **B-roll density**: per channel signature (typically 40–60% b-roll coverage on authority videos)
- **Cut density**: matches the channel's editing rhythm
- **Music**: under-score throughout; shifts at section transitions
- **On-screen text**: section titles, key statistics, callout boxes for retention boosts
- **Pattern-interrupts**: every 45–60s minimum

## Verification checklist (pre-publish)

- [ ] Hook density ≥ 3 mechanisms in first 30s (INV-11)
- [ ] Loop opened by 0:08; closed before 50% mark
- [ ] Pattern-interrupt cadence ≥ every 60s
- [ ] Each section has re-hook → story → lesson pattern
- [ ] Banned vocabulary scan passed (INV-7)
- [ ] Title-thumbnail-promise alignment (INV-10)
- [ ] Predicted AVD ≥ channel bottom-quartile band (INV-9)
- [ ] Mode declared: authority
- [ ] Awareness level declared
- [ ] Compartments cited
