# Productivity & Career — Vertical Playbook

> Vertical-specific playbook for productivity / career YouTube channels (knowledge-work systems, career-development, productivity tools, professional skill-building). Read alongside `reference/benchmarks/productivity-career.md`.

## Vertical positioning advice

Productivity / career is a steady-build, system-loving, knowledge-worker niche. Positioning gates:

1. **A demonstrable working system.** The operator has an actual productivity system in active use, with the artifacts (templates, dashboards, workflows) to show. Generic productivity advice loses to operators with visible systems.

2. **A specific tool or methodology.** "Productivity" is too broad. "Notion for project management," "Obsidian for second-brain," "Linear for product teams," "GTD for executives" — specificity wins.

3. **An audience-tier match.** Beginner knowledge workers, intermediate operators, scaling executives — each tier has different system needs. The channel commits to a tier or explicitly bridges across tiers.

4. **Update discipline.** Productivity tools update frequently. Stale tutorials become liabilities.

5. **Quality of the artifact.** Templates, vaults, workflows the channel ships should be production-quality. The audience compares against operator-tier productivity content; cheap templates lose.

## Typical offer structures

- **Self-paced courses on systems / tools** — $97–$1,500; the dominant offer category
- **Templates / template packs** — $30–$500; ancillary; lead-magnet-bridge-strong
- **Group cohort courses** — $497–$3,000; multi-week structured programs
- **One-on-one consulting** — $200–$1,500+ per hour; selective
- **Membership / community** — $20–$100/mo; ongoing template / system access
- **Done-for-you system installation** — $2K–$10K; for executives / teams

OS bullseye: creators with courses / cohorts in the $500–$3,000 band, lead-magnet → email → tripwire → core architecture works well.

## Typical audience awareness distribution

- Unaware: 10–15%
- Problem-aware: 25–35%
- Solution-aware: 30–40%
- Product-aware: 15–20%
- Most-aware: 5–10%

Productivity audiences typically arrive problem-aware ("I need a system"). The channel's role is to introduce the operator's solution-aware framing and product-aware mechanism.

## Channel architecture archetypes that work

### The productivity-systems-builder architect

The dominant archetype. Channel demonstrates the operator's actual system. Foundational playlist establishes the system's foundations. Pillar playlists per system component (capture, processing, organization, review). Library compounds as a system-build curriculum.

### The tool-stack-curriculum architect

Channel organized around a specific tool. Notion-channel, Obsidian-channel, Linear-channel. Foundational playlist introduces the tool. Pillar playlists deepen specific use-cases. Library compounds as a tool-specific curriculum.

### The career-developer architect

Channel positions on career skills (interview, leadership, communication, professional skill-building). Foundational playlist establishes the channel's career philosophy. Pillar playlists per career stage / skill domain.

## Content pillars typical for this niche

Recommended 4–5 pillars:

1. **System foundations** (20–25%) — the operator's core productivity system
2. **Tool tutorials / walkthroughs** (20–25%) — specific tool use
3. **Workflow case studies / examples** (15–20%) — applied demonstrations
4. **Career skill development** (15–20%) — transferable skills
5. **Tool reviews / comparisons** (10–15%) — contrast-format / comparative

## Voice of viewer in this niche

Recurring pain language:

- "I have so many tools but I can't find anything"
- "I keep starting and stopping systems"
- "I'm overwhelmed by [tasks / projects / inputs]"
- "I want to be more productive but I don't know how"
- "My team can't [collaborate / coordinate / track]"
- "I'm [stressed / behind / drowning]"
- "I don't have time to [build a system]"

## Conversion path specific to productivity / career

Lead-magnet → email → tripwire → core typical. Pinned VSL secondary.

For lead-magnet bridge:
- High-conversion magnets are downloadable templates / vaults / workflows
- Email sequence walks the subscriber through implementing the system
- Tripwire ($30–$200) is a polished template pack or a single-tool course
- Core ($497–$2,000) is the comprehensive course / cohort

For pinned VSL (when used):
- Length: 25–40 minutes (shorter than business; the audience is more transactional / less identity-driven)
- Hook: system-specific or efficiency-promise
- Mechanism: visible system in operation
- Price anchoring: time-savings vs course-cost framing

## Common failure modes specific to this niche

### Stale tool versions

**Failure**: tutorials using old tool versions; misleading the audience.
**Prevention**: update discipline. Annual library audit. Pinned-comment update notes for outdated content.

### System without artifact

**Failure**: teaching a system without shipping the artifact (template, vault). Audience can't implement.
**Prevention**: artifact-first publishing. The system is downloadable.

### Generic system advice

**Failure**: "be more focused" / "single-task" / "use a calendar" — content commoditized by every productivity channel.
**Prevention**: operator-specific systems with operator-specific artifacts. The system is the IP.

### Tier-mismatch

**Failure**: complex executive systems offered to beginner audience. Overwhelming.
**Prevention**: tier-tagged content. Beginner content for beginners; advanced for advanced.

### Aspiration-without-implementation

**Failure**: showing beautiful systems the audience aspires to but cannot implement. Dispiriting.
**Prevention**: implementation-focused content. "Start with X; in 6 weeks, add Y."

- **Affiliate disclosure**: INV-15 + tool-affiliate-relationship transparency
- **Earnings / career claims**: substantiated only

## Channel-stage progression

Productivity / career channels typically progress through:

- **Months 0–6**: niche-finding; tool / system focus; library 10–30 videos
- **Months 6–18**: artifact-library development; tripwire offers launch; library 30–75 videos
- **Months 18–36**: cohort offers launch; library 75–150 videos; long-tail traffic strong
- **Years 3+**: tool-stack authority; library at 150+ videos; courses + cohorts at scale

The library compound is the dominant lever. Productivity tutorials are evergreen-strong; operators with deep tool-stack curricula compound for years.
