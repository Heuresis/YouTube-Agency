# Title Specimen 036 — Question-as-title

## Structure (paraphrased)
"Is [Common Assumption] Actually True?"

## Worked structural examples (anonymized)
- "Does [Common Practice] Actually Work?"
- "Is [Conventional Wisdom] Right?"
- "Are You [Common State] — Or Just [Alternative State]?"

## Why this works (the imitation lesson)
Question-as-title formats work because they invite the viewer to participate in the answer rather than receive a verdict — this triggers a more engaged click. "Actually" is load-bearing; it signals the answer will be non-obvious. The format is highest-CTR when paired with a thumbnail that visually answers the question (e.g., a red X or a green check), which compresses the resolution.
