# Thumbnail Specimen 010 — Warning / alert visual frame

## Structure (paraphrased)
- Visual: A high-saturation red color element (border, banner, exclamation icon, warning sign). The operator's face visible in supporting position with a serious / concerned expression.
- Text overlay: All-caps text starting with "WARNING," "STOP," or "DON'T."

## Why this works (the imitation lesson)
The warning / alert thumbnail uses red-color and warning-language to interrupt the scroll. The operator's serious expression supports the warning frame; smiling expressions undercut the urgency. The structural rule: the body must deliver actionable advice, not generic doom — using warning visuals without warning content burns audience trust.

## When to use
Reactive / advisory content where there's a real risk to highlight. Loses effectiveness rapidly when over-used.
