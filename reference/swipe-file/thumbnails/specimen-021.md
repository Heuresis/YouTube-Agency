# Thumbnail Specimen 021 — Framework-diagram thumbnail

## Structure (paraphrased)
- Visual: Simplified framework visualization — boxes + arrows, pyramid, matrix, or flowchart. Clean, geometric.
- Text overlay: Framework name in large bold, often single-color. Sometimes "[Name] FRAMEWORK" or "MY [N]-STEP SYSTEM".
- Color: Limited palette (often 2–3 colors), professional-looking.

## Why this works (the imitation lesson)
Framework-diagram thumbnails work because they signal structured intellectual content — the audience knows what they're getting (a teaching, not a story). The framework name as text overlay creates a memorable handle that becomes searchable and shareable. The clean visual style filters for serious viewers and elevates the channel's perceived production quality.

## When to use
- mode: authority
- audience awareness: most-aware
- pairs with: numbered-framework title, bound-promise hook
- channel stage: 100K–1M
