# Thumbnail Specimen 002 — Shocked-face + bold-number

## Structure (paraphrased)
- Visual: The operator's face occupying ~40% of the frame with a shocked / surprised / wide-eyed expression. Background can be neutral or contextual.
- Text overlay: A specific bold number ($100K, 10M, 90 DAYS) in a contrasting color, sized to dominate the remaining frame.

## Why this works (the imitation lesson)
The shocked-face thumbnail exploits emotional contagion — the viewer's brain mirrors the operator's surprise, generating curiosity about what could produce that reaction. The bold number is the specific anchor; without the number, the face reads as generic clickbait.

## When to use
Single-event-anchored content. The number must reference a specific event the body delivers; mismatched thumbnail-number / body-content produces backlash.
