# Thumbnail Specimen 003 — Red-circle + arrow data callout

## Structure (paraphrased)
- Visual: Image of a specific artifact (analytics dashboard, social media metric, screenshot). A bright red circle drawn around a specific data point. A red arrow pointing at the circled element.
- Text overlay: 1–3 words that direct attention ("LOOK AT THIS," "$100K WEEK," "27% CTR").

## Why this works (the imitation lesson)
The red-circle thumbnail uses high-saturation visual annotation to direct attention to a specific artifact. The viewer's eye is drawn to the circle, then to the arrow, then to the text. The structural rule: the artifact must be real and specific; faked metrics produce immediate distrust.

## When to use
Data-driven / case-study content where the operator has a specific verifiable artifact to feature. Less effective for narrative / non-numeric content.
