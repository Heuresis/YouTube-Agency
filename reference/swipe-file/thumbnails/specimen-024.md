# Thumbnail Specimen 024 — Operator-with-artifact thumbnail

## Structure (paraphrased)
- Visual: Operator on left holding or pointing to a tangible artifact — a printed document, screen, notebook, contract, dashboard. Artifact is on right, partly obscured but recognizable.
- Text overlay: 3–5 words referencing the artifact ("MY EXACT TEMPLATE" or "THE FRAMEWORK"). Bold.
- Color: 2–3 colors with the artifact slightly emphasized.

## Why this works (the imitation lesson)
Operator-with-artifact thumbnails work because the artifact is the value proposition made visible — the audience can see what they'll get if they click. The "exact" or "actual" framing in text overlay is load-bearing; it differentiates from generic "I'll explain" thumbnails. The artifact's partial visibility (recognizable but not fully readable) is intentional curiosity-density.
