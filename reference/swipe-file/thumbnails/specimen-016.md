# Thumbnail Specimen 016 — Property-with-receipt thumbnail

## Structure (paraphrased)
- Visual: Property exterior photo as the main background. Inset (top-left or bottom-right) of operator's face.
- Text overlay: Specific dollar figure or terms — "$0 DOWN" or "+$1,200/MO" or "240% ROI".
- Color: Photo's native colors + one accent for the dollar callout (often green or red-then-green).

## Why this works (the imitation lesson)
Property-with-receipt thumbnails work because the property is concrete proof — the audience can see what the deal looks like. The operator inset is a parasocial anchor; without it, the thumbnail loses authority attribution. The dollar-figure callout is load-bearing — vague claims like "GREAT DEAL" don't trigger evaluation, "$0 DOWN" does.

## When to use
- mode: authority
- audience awareness: solution-aware
- pairs with: deal-as-proof title, PAST-formula hook
- channel stage: 10K–100K, 100K–1M
