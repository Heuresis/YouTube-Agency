# Thumbnail Specimen 017 — Aesthetic-flatlay thumbnail

## Structure (paraphrased)
- Visual: Top-down (flatlay) of a minimalist desk — notebook, single tool / object, soft natural light.
- Text overlay: 3–5 words in clean sans-serif. Often centered.
- Color: Muted neutral palette + 1 accent (a colored notebook, a single warm object).

## Why this works (the imitation lesson)
Aesthetic-flatlay thumbnails signal a specific lifestyle aspiration — the audience associates the visual with the version of themselves they want to become. The flatlay format is uncluttered, which contrasts with the maximalist YouTube norm and stands out. It works specifically in productivity / career niches because the audience already values minimalism aesthetically.
