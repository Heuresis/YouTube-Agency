# Thumbnail Specimen 009 — List-number bold callout

## Structure (paraphrased)
- Visual: A specific list number (e.g., "7", "15", "3") sized very large to dominate the frame. The operator visible in supporting position.
- Text overlay: 2–4 words completing the list-frame ("[Number] [Things] [Outcome]").

## Why this works (the imitation lesson)
The list-number thumbnail signals immediately that the video is a discrete-list video. The very large number is the load-bearing visual — viewers should be able to read the number from a thumbnail at half-size on mobile.

## When to use
Listicle / compilation videos paired with the number-of-things title pattern (title specimen 011). Less effective for non-list videos.
