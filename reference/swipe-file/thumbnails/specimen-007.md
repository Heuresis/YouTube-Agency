# Thumbnail Specimen 007 — Bold-claim + receipt

## Structure (paraphrased)
- Visual: A specific dollar / metric figure prominently displayed. A second smaller element showing a verifying artifact (a Stripe dashboard, an analytics screenshot, a bank statement) in supporting position.
- Text overlay: The dollar / metric figure repeated as text.

## Why this works (the imitation lesson)
The bold-claim + receipt thumbnail pairs the impressive number with a verifying artifact in a single visual. The viewer can see both the claim and the proof in one glance — a structural shortcut that converts impression-skeptics into watchers.

## When to use
Authority-mode videos with specific verifiable revenue / outcome claims. Most effective when the operator can show real artifacts.
