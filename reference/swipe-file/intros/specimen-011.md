# Intro Specimen 011 — Sensory-specific event-anchor intro

## Structure (paraphrased)
- Beat 1 (15–25s): Specific event with timestamp. "It was [specific date / time], and I had just [specific failure / breakthrough event]."
- Beat 2 (25–40s): Emotional state at the event. "I was [emotional state — anxious / excited / desperate]. I had no idea this moment would [reframe]."
- Beat 3 (40–55s): The pivot the event produced. "What happened next changed everything. [Brief description of the pivot]."
- Beat 4 (55–75s): Generalize to the audience's potential transformation. "And in this video, I'm going to show you exactly how to recognize and engineer your own [pivot moment]."

## Why this works (the imitation lesson)
The event-anchor intro uses sensory specificity (date, time, emotional state) to create a vivid scene. The structural rule: the specifics must be real and verifiable — fabricated specifics produce a "this feels off" sensation that audiences detect even if they can't articulate why.

## When to use
Storytelling-tilted videos where the operator has a specific real event to anchor. Less suitable for purely-tactical content where the event-frame would feel forced.
