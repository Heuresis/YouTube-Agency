# Intro Specimen 012 — Framework-claim brand-equity intro

## Structure (paraphrased)
- Beat 1 (15–25s): State the framework's name + its function. "I call this the [framework name] — the [N]-phase system that [specific outcome]."
- Beat 2 (25–45s): Position the framework against alternatives. "Most people teach [common alternative] — and it works to a point. But the [framework name] goes further by [specific differentiator]."
- Beat 3 (45–60s): Establish the operator's standing. "I've used this with [N students / over $X in revenue / across N years], and the pattern is consistent."
- Beat 4 (60–80s): Outline the framework's structure. "The [N] phases are: [phase 1 name], [phase 2 name], [phase 3 name], and [phase 4 name]."
- Beat 5 (80–90s): Promise the body. "In the next [X] minutes, I'll walk you through each phase with specific examples."

## Why this works (the imitation lesson)
The framework-claim intro converts the video into a brand-equity asset. By naming the framework, naming each phase, and consistently using these names across multiple videos, the operator builds equity in the framework itself. Audiences begin to refer to "the [framework name]" by name in comments, search queries, and DMs.

## When to use
Authority-mode videos where the operator wants to build long-term equity in a named system. Most effective when the framework name is short, memorable, and distinctive.
