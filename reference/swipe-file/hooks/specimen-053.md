# Pattern-Interrupt Specimen 053 — Standard pattern-interrupt grammar

## Structure (paraphrased)
- Type 1 — Visual cut (every 45–60s): Hard cut to different angle, location, or B-roll. Resets the viewer's visual processor.
- Type 2 — Verbal pivot (every 90s): "But here's where it gets interesting..." or "Now, before I show you the next part..."
- Type 3 — Text overlay (every 30–60s): On-screen text emphasizes a single phrase from speech, color-coded.
- Type 4 — Pace shift (every 2 min): Speaker visibly slows down for emphasis, then accelerates.
- Type 5 — Question to viewer (every 90s): "Does this make sense so far?" or "Are you with me?"
- Type 6 — Callback to opener (1–2 times per video): "Remember the [thing] I mentioned at the start? Here's why that matters now."

## Why this works (the imitation lesson — 2–4 sentences)
Pattern-interrupts solve the retention-decay problem — without intervention, viewer attention drops at predictable intervals (60s, 3min, 5min). The six interrupt types operate on different cognitive channels (visual, verbal, textual, kinesthetic, social, narrative) so they can stack without feeling redundant. The callback (Type 6) is the structurally most powerful because it converts the entire video into a closed loop.
