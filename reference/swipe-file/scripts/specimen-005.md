# Script Section Specimen 005 — Comparative-decomposition body section

## Structure (paraphrased)
- Beat 1 (10–20s): State the comparative axis. "Let's analyze [option] by [specific metric]."
- Beat 2 (20–60s): Walk through specific data for option A. "First, the [metric]. [Option A] pays [specific dollar / percentage]. That means if you [generic effort], you'd earn [specific outcome]."
- Beat 3 (60–120s): Walk through specific data for option B. "Now [option B]: [metric]. [Specific dollar / percentage]. Same effort earns [specific outcome]."
- Beat 4 (120–160s): Explicit calculation. "That's a [specific multiple] difference for the exact same [unit of work]."
- Beat 5 (160–220s): Identify the structural reason for the gap. "The reason for the gap is [structural difference between the two options]."

## Why this works (the imitation lesson)
The comparative-decomposition section converts an opinion ("X is better") into a quantified analysis. The structural rule: every claim must be paired with specific numbers. Generic "X earns more" is opinion; specific "X earns 6x more for the exact same effort" is analysis.

## When to use
Comparison-format videos. Especially powerful in adjacent-niche videos where the audience is already evaluating both options.
