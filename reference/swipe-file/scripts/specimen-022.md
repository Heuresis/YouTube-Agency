# Script-Section Specimen 022 — Side-A-vs-Side-B body section

## Structure (paraphrased)
- Re-hook (10–15s): "OK — let's see what happens when we run [Side A] versus [Side B] head-to-head."
- Side A walkthrough (90–120s): Explain the approach. Show what it looks like in practice. Visual on screen.
- Side A results (30–45s): Numbers / outcome. On-screen.
- Side B walkthrough (90–120s): Same structure as Side A. Equal time, equal visual treatment.
- Side B results (30–45s): Numbers / outcome.
- Comparison (45–60s): Direct head-to-head graphic. "Here's the comparison."
- Synthesis (30–45s): What the comparison teaches. "The reason [winning side] won is [structural reason]."
- Transition (10–15s): "So what does this tell us about how to make this decision?"

## Why this works (the imitation lesson — 2–4 sentences)
Side-A-vs-Side-B sections earn retention because the comparison structure is itself the value proposition — viewers stay to see the result. Equal time and equal visual treatment of both sides (Beats 2–5) is the load-bearing fairness signal; if one side gets short shrift, viewers detect bias and disengage. The synthesis beat (Beat 7) elevates the comparison from horse-race to teaching.
