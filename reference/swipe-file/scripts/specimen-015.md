# Script-Section Specimen 015 — Source-citation body section

## Structure (paraphrased)
- Re-hook (15–20s): Pose the section's philosophical question. "Where does this idea come from, and why has it survived this long?"
- Source introduction (45–60s): Introduce the canonical source. "[Historical thinker / text] wrote about this in [period]. The core line — translated, paraphrased: '[paraphrased idea].'"
- Modern translation (60–90s): Translate the canonical line into modern context. "What [thinker] meant in [period] was [interpretation]. In your life, this looks like [specific example]."
- Application (90–120s): Show what the principle requires of the viewer. "To apply this, you need to [specific behavior change]. Here's what that looks like in practice."
- Failure mode (30–45s): Pre-empt the misapplication. "The most common way this gets misapplied is [specific failure]. Avoid that by [specific check]."
- Transition (10–15s): Bridge. "OK — that's the principle. Now let's stack it with [next idea]."

## Why this works (the imitation lesson — 2–4 sentences)
Source-citation sections earn authority by transferring the gravity of canonical knowledge — the creator doesn't need to defend the idea, only translate it. The modern-translation step is the load-bearing element; without it, the citation feels like name-dropping rather than teaching. The failure-mode pre-empt (Beat 5) is what differentiates teaching from quoting — it shows the creator has actually used the principle.
