# Script Section Specimen 004 — Phase-by-phase framework body section

## Structure (paraphrased)
- Beat 1 (10–20s): State the phase number and name. "This is Phase [N] — [phase name]."
- Beat 2 (20–40s): State what the phase covers + its purpose. "[Phase name] is all about [specific function]. The reason this matters is [mechanism]."
- Beat 3 (40–80s): Walk through specific timeline / quantities. "For my [specific event], I started [phase] exactly [specific time] before [next milestone]. Timing matters — too long and [bad outcome]; too short and [bad outcome]."
- Beat 4 (80–140s): Specific tactical examples. "Instead of [generic action], I did [specific action]. For example: [specific worked case with detail]."
- Beat 5 (140–200s): Explain the psychological / mechanism reason. "When [target audience's brain] sees [specific stimulus], it [specific response]. It's like [analogy]. That's why [tactic works]."
- Beat 6 (200–240s): Bridge to the next phase. "[FOMO / curiosity / consequence reference]. That's where the next phase comes in."

## Why this works (the imitation lesson)
The phase-by-phase section structure is the canonical "X-step framework" body pattern. Each phase repeats the structure: name → function → timeline → tactics → mechanism → bridge. Beat 5's mechanism explanation is the load-bearing element — without it, the phase reads as opinion ("do this because I said so"); with it, the phase reads as principle ("do this because of how human cognition works").

## When to use
Any video built around a multi-phase named framework (hook specimen 024, intro specimen 012). Most effective when each phase has a distinctive name and a clear function.
