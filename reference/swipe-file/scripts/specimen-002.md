# Script Section Specimen 002 — BUT-THEREFORE narrative engine in body

## Structure (paraphrased)
- Beat 1 (10–20s): Set the scene with sensory specificity. "It was [time/date]. I was [location]. I had just [specific event]."
- Beat 2 (20–60s): Introduce the conflict using BUT. "[Specific narrative beat]. [Another narrative beat]. **BUT** [reversal that destabilizes the status quo]."
- Beat 3 (60–120s): Show the consequences using THEREFORE. "**THEREFORE**, [specific consequence]. [Detail of consequence]. [Emotional response of character]."
- Beat 4 (120–180s): Introduce a second reversal using BUT. "**BUT** then [unexpected turn]. [Specific moment of insight or change]."
- Beat 5 (180–240s): Show the new consequence using THEREFORE. "**THEREFORE**, [specific transformation outcome]. [Detail]. [Generalization for the audience]."

## Why this works (the imitation lesson)
The BUT-THEREFORE engine prevents the failure mode where stories collapse into "and then... and then..." sequences. Each BUT creates a reversal that re-engages attention; each THEREFORE creates a consequence that pays off the previous reversal. The structural rule: a story segment with no BUTs is structurally a list and will not engage.

## When to use
Every personal-story segment, case-study segment, and narrative example in any long-form video. The mechanism is universal across genres but particularly load-bearing in coaching / education content where stories drive belief installation.
