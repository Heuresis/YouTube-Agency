# Contrast-Format Specimen 011 — Tool A vs Tool B in production format

## Structure (paraphrased)
- Frame (0:00–0:30): "I've used both [Tool A] and [Tool B] in production for [N months each]. I'm going to tell you what nobody else will."
- The shared use case (1–2 min): Define what both tools claim to do. Specific use case.
- Tool A in production (2–3 min): What it's actually like to use day-to-day. Specific gripes and wins.
- Tool A — what it's best at (1 min): Honest assessment.
- Tool B in production (2–3 min): Same structure.
- Tool B — what it's best at (1 min): Honest assessment.
- The deciding factors (1–2 min): The variables that determine which is right for which team.
- The verdict for [most common context] (1–2 min): "If you're [common audience descriptor], pick [tool]. Here's why."
- The migration cost (1 min): "If you're already on [losing tool], the switching cost is [honest assessment]."

## Why this works (the imitation lesson — 2–4 sentences)
Tool-comparison contrast formats earn solution-aware-audience retention because the audience is mid-decision and wants honest comparison. The "I'm going to tell you what nobody else will" framing (Beat 1) signals editorial integrity. The migration cost beat (Beat: migration cost) is the load-bearing trust element; pretending switching is free is the most common credibility mistake in this format.
