# Contrast-Format Specimen 016 — N people, same goal, different paths

## Structure (paraphrased)
- Frame (0:00–0:30): "I followed [N] people with the same goal for [timeframe]. They all took different paths. Some succeeded, some didn't."
- Subject 1 — start, path, outcome (2–3 min): The arc.
- Subject 2 — start, path, outcome (2–3 min): Same.
- Subject 3 — start, path, outcome (2–3 min): Same.
- Subject N — start, path, outcome (2–3 min): Same.
- Side-by-side outcome comparison (1–2 min): Who succeeded, who didn't.
- The pattern that separated success from failure (1–2 min): "The one variable that mattered most was [specific factor]."
- The variable that didn't matter (1 min): "I expected [common variable] to matter — it didn't."
- Application (1–2 min): "If you're pursuing this goal, focus on [variable that mattered]. Don't waste time on [variable that didn't]."

## Why this works (the imitation lesson — 2–4 sentences)
Multi-subject peer-comparison contrast formats earn retention because they offer pattern-recognition value the audience can't get from single case studies — the comparative element is the insight. The "variable that didn't matter" beat (Beat: variable that didn't) is the load-bearing differentiator; it explicitly de-prioritizes commonly-emphasized factors, which is intellectually generous. The application beat translates the pattern into prescription.
