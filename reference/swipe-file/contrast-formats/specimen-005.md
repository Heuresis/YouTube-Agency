# Contrast-Format Specimen 005 — $X vs $XXX vs $XXXX format

## Structure (paraphrased)
- Frame (0:00–0:30): "I tested [Tier 1 budget], [Tier 2 budget], and [Tier 3 budget] versions of [thing] to find out where the price-quality curve actually breaks."
- Tier 1 walkthrough (2–3 min): The cheapest option. What you get. Where it falls short. Specific test outcomes.
- Tier 2 walkthrough (2–3 min): The mid-tier option. Same structure.
- Tier 3 walkthrough (2–3 min): The premium option. Same structure.
- Direct comparison (1–2 min): Side-by-side performance graphic.
- The price-quality curve (1–2 min): "The interesting thing is the curve isn't linear. Here's where the inflection points are."
- Recommendation by use-case (1–2 min): "If you're a [use case A], get [tier]. If you're [use case B], get [tier]."
- The honest meta-lesson (45–60s): "What I actually learned from this is [lesson about price-quality in this domain]."

## Why this works (the imitation lesson — 2–4 sentences)
Budget-tier contrast formats earn retention because they convert the audience's purchasing decision into a research project that's already been done. The price-quality curve framing (Beat 6) is the load-bearing element; without it, the format reads as three separate reviews. Recommendation by use-case (Beat 7) prevents the video from being interpreted as an absolute ranking, which is more useful and more honest.
