# Contrast-Format Specimen 025 — Same input, different outputs comparison

## Structure (paraphrased)
- Frame (0:00–0:30): "I gave the same [input: brief / data / starting state] to [N people / methods / approaches]. The outputs were unrecognizable from each other."
- The shared input (1–2 min): Define exactly what was held constant.
- Output 1 — process and result (2–3 min): How they approached it. What they produced.
- Output 2 — process and result (2–3 min): Same.
- Output N — process and result (2–3 min): Same.
- Side-by-side outputs (1–2 min): Visual comparison.
- The variable that determined output (1–2 min): "What separated the outputs wasn't [obvious variable]. It was [counterintuitive variable: experience, frame, taste, decision-rule]."
- The teachability question (1–2 min): "Can the variable that mattered be taught? Yes — here's how."
- Application (1 min): "If you want to produce outputs more like [winning output], the leverage is in [specific practice]."

## Why this works (the imitation lesson — 2–4 sentences)
Same-input-different-outputs contrast formats earn retention because they isolate the operator-variable from the input-variable, which is the more interesting question. The "variable that determined output" beat (Beat: variable that determined output) is the load-bearing insight; if it's just "talent" or "experience" the video is depressing, but if it's a teachable practice or stance, the format becomes high-leverage education. The teachability question converts insight into hope.
