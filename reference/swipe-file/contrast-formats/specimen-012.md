# Contrast-Format Specimen 012 — Protocol A vs Protocol B body-test format

## Structure (paraphrased)
- Frame (0:00–0:30): "I ran [Protocol A] for [N weeks], then [Protocol B] for [N weeks]. Same diet, same sleep, only the protocol changed."
- The variables held constant (1 min): Proof of methodological honesty.
- Protocol A — what it is (1–2 min): The protocol design.
- Protocol A — daily reality (2 min): What it's like to actually run it.
- Protocol A — measured outcomes (1–2 min): Specific metrics with on-screen data.
- Protocol B — what it is (1–2 min): Same structure.
- Protocol B — daily reality (2 min): Same.
- Protocol B — measured outcomes (1–2 min): Same.
- Direct comparison (1–2 min): Side-by-side metrics.
- The verdict (1–2 min): Which won, by how much, and the structural reason.
- N-of-1 caveat (45–60s): "This is one person, one trial. Here's how to think about it for your context."

## Why this works (the imitation lesson — 2–4 sentences)
Protocol-comparison contrast formats earn evidence-led-audience retention because the controlled variables (Beat 2) signal scientific rigor that's rare in fitness content. The N-of-1 caveat (final beat) is the load-bearing trust element; pretending one trial is universal evidence is the most common credibility mistake in this format. Specific measured outcomes outperform self-reported impressions.
