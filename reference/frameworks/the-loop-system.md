# The Loop System (Curiosity Loop Architecture)

## Origin archetype
Senior YouTube script consultant who reverse-engineered the retention curves of educational creators with 7-figure businesses.

## What it solves
Viewers leave when their working memory closes a question. Open questions ("loops") create cognitive tension that demands resolution and pulls viewers through the natural drop-off zones at 30s, 2min, and 5min.

## Structure — five loop types

| Loop type | Pattern | Generic structural example |
|---|---|---|
| Secret loop | Tease hidden information | "There's one [thing] I've never shared publicly..." |
| Mistake loop | Tease an error the viewer is making | "You're making a mistake costing you 50% of [outcome]..." |
| Result loop | Tease an outcome without revealing the method | "This strategy got me [specific result] in [specific timeframe]..." |
| Discovery loop | Tease an unexpected finding | "I analyzed [large number] of [things] and found something shocking..." |
| Callback loop | Reference an earlier moment | "Remember when I said [X]? There's more to it than I told you..." |

## Structure — five placement positions (mapped to a 10-minute video)

| Position | Timecode | Action |
|---|---|---|
| Opening | 0–30s | Open 2–3 primary loops |
| Early body | 30s–2min | Reference loops without closing them |
| Mid-video | 3–5min | Close minor loops, maintain major ones |
| Late body | 6–8min | Final tease before payoff |
| Climax | 8–10min | Close ALL remaining loops |

## Hard rules
1. Maximum 3 major loops per video — more than 3 overloads working memory and the audience disengages.
2. ALWAYS close every loop by video end. An unclosed loop is a betrayal that shows up as a comment-section complaint and a subscriber loss.
3. Close major loops at 70–80% through the video — not earlier (kills retention to outro) and not later (rushed payoff).
4. Each loop must have clear, specific value — vague "something interesting" loops feel like clickbait and fail.

## Loop budget by video length
- 8–12 minute video: 2–3 major loops + 3–5 minor loops
- 15–20 minute video: 3–4 major loops + 5–7 minor loops

## When to use
Every long-form educational or persuasion video. Live demos can use loops more sparingly (the demo itself is its own loop).

## Why it works
The Zeigarnik effect — the brain allocates more cognitive resources to incomplete tasks than completed ones. An open loop is an incomplete task; the brain stays engaged because it cannot file the question away.

## Variants observed
Some operators teach 3 loop types; others teach 7. The canonical 5 types retained here are the minimum-set that covers the full distribution of loop-creation patterns observed across the source material.
