# Million-View Framework — Structural Decomposition

## Origin archetype
Educational-content specialist operating a comprehensive system for online-coach creators that combines deep audience research, strategic hook writing, proven video structures, and psychological retention techniques.

## What it solves
Instead of treating "going viral" as luck, the framework decomposes it into deterministic components that consistently produce 7-figure-view educational content.

## Structure — the seven decomposable layers

1. **Video ideation system** — competitor intelligence gathering + content DNA extraction (transcripts of top-3 viral videos in niche) + AI-powered audience psychology mining + unique angle development.
2. **Engagement heat-map method** — timeline scrubber analysis on top-3 videos, click on every "replay hotspot," document content at hotspots, identify patterns across videos.
3. **Comment intelligence** — analyze comments for emotional triggers, "mind-blown" reactions, common complaints, gaps in existing content, repeated questions.
4. **Title formula library** — 5 high-CTR formulas (Curiosity Gap, Bound Promise, Contrarian Angle, Social Proof Stack, Universal Hook) + the 4-pillar split.
5. **Hook framework** — the 4 hook formulas (PAST/AID/Contrast/Statistic) + the universal hook success formula.
6. **Body structure** — the re-hook → story → lesson per-section pattern + 7 transition types + curiosity loops.
7. **Outro and binge loop** — the 3-step formula (Link Back → New Problem → Next Video Promise) + soft / hard / 90-second CTA strategy.

## When to use
As the macro-organizing scaffold for any creator-channel system. The seven layers correspond to the production phases (idea → research → title → hook → body → transitions → outro) of every video.

## Why it works
It explicitly maps the decision points in long-form video creation to the cognitive mechanisms in the viewer's brain. Each layer addresses a distinct viewer-attention mechanism; missing any layer produces a predictable failure mode (e.g., missing layer 2 produces topic-blind videos that don't outsophisticate competitors).

## Variants observed
Some operators teach a 4-layer simplified version (idea / hook / body / outro); others teach a 12-layer extended version (adding sub-layers for thumbnails, scripting, B-roll, etc.). The 7-layer form retained here is the minimum-set that captures every load-bearing decision point.
