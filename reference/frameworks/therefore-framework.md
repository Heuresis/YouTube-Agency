# The THEREFORE Framework

## Origin archetype
Direct-response-trained script consultant porting the THEREFORE-style logic chain into long-form video persuasion.

## What it solves
A more compact application of Logic Ladders. Where logic ladders use ↓ between every step, the THEREFORE framework uses an explicit "therefore" connective at the conclusion of a multi-clause causal chain.

## Structure — the THEREFORE chain

"[Step 1]. [Step 2]. [Step 3]. [Step 4]. **THEREFORE: [Conclusion that follows from the chain].**"

## Generic structural example
"You post videos with proper frameworks. They generate qualified traffic. Qualified traffic watches your content. They click your link in description. They land on a converting page. **THEREFORE: posting one good video earns more revenue than posting ten generic videos.**"

## When to use
When stating a claim that the audience would otherwise dismiss as unsupported. The THEREFORE chain forces the audience to either accept the conclusion (because each preceding step is plausible) or reject the entire chain (which requires effort).

## Distinction from Logic Ladders
Logic Ladders use multi-step chains with explicit ↓ visualization; THEREFORE Framework is the prose version embedded in a single paragraph. Logic Ladders are typically on-screen graphics; THEREFORE Framework is typically narrated.

## Why it works
Same mechanism as Logic Ladders — the brain's default move is to dismiss conclusions, not individual steps. THEREFORE chains weaponize the steps to make dismissal effortful.

## Variants observed
Some operators use BECAUSE chains (causation only); others use SO chains (consequence only). THEREFORE retained because it handles both causation and consequence and reads as natural in spoken voice.
